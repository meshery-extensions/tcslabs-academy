Challenges

Hands-on challenges to test and validate your AI-native infrastructure skills with Meshery. Each challenge pairs a guided lab with a graded exam.
1 minute read

Challenges 🔗

Build a Meshery MCP Tool

Build a small Model Context Protocol tool that exposes live Meshery and cluster state to a coding agent. Start read-only, back it with the Meshery API, and connect it to an agent to answer questions about your infrastructure.

CAINA Capstone - Generate, Deploy & Validate a Design with an Agent

The hands-on capstone for the Certified AI-Native Infrastructure Associate (CAINA). Use an LLM to generate a Meshery design from intent, deploy it with a coding agent and mesheryctl, and validate it in Meshery.

CAINP Capstone - Operate an AI Workload with Agents & Meshery

The end-to-end capstone for the Certified AI-Native Infrastructure Professional (CAINP). Deploy and operate an inference workload with Meshery, wire up observability, build an MCP tool for an agent, and run an agent-assisted incident-and-remediation loop under governance.

Heal Meshery

Diagnose and remediate a broken deployment with an LLM and Meshery. Break a workload, use a coding agent to find the root cause from real signals, propose a fix behind an approval gate, and confirm recovery.

Ship It with an Agent

Use a coding agent and Meshery to deploy a microservices application from a design - the warm-up challenge for the CAINA credential. Generate or import a design, deploy it with an agent, and verify the rollout.
