Foundations of Cloud Native Management with Meshery

Start here. Build a working mental model of cloud native and Kubernetes, install Meshery and connect a cluster, and learn to capture infrastructure as reusable Meshery designs. The grounding for everything else in the academy.
1 hours read
1. Cloud Native & Kubernetes Essentials

Build the foundational vocabulary and mental models for cloud native infrastructure - containers, Kubernetes, declarative APIs, and GitOps - that every subsequent course in this learning path depends on.

2. Meshery Essentials

Get hands-on with Meshery: install it, connect a cluster, and understand the core concepts that underpin every workflow in the academy - from visual design to AI-assisted operations.

3. Infrastructure as Designs

Learn how Meshery designs capture infrastructure intent as version-controlled, shareable artifacts - and how to import, inspect, deploy, and collaborate on them using Kanvas, the Catalog, GitHub, and Meshery’s environment model.
