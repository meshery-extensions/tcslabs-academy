Embeds a PDF file in a responsive iframe.

```text
{{</* hextra/pdf "sample.pdf" */>}}
```

**Parameters:**

| Parameter | Description | Default |
|-----------|-------------|---------|
| positional | Path to the PDF file (page resource, asset, or absolute path) | _(required)_ |

**Example:**

{{< hextra/pdf "sample.pdf" >}}
